## <PE 파일>
`PE 파일` : (=Portable Executable File)

            = Windows 환경의 실행 파일 포맷

Portable(=가지고 다닐 수 있음)

<특징>
- 플랫폼에 독립적

<종류>
1) 실행 파일 계열 : EXE, SCR
2) 라이브러리 계열 : DLL, OCX
3) 드라이버 계열 : SYS
4) 오브젝트 파일 계열 : OBJ

 = 실행가능한 파일 + 오브젝트 파일

<PE 파일 헤더 분석> = (PE header)
- PE 파일 헤더 : 파일이 실행되기 위해 필요한 모든 정보가 적혀있음
    Ex) 어떻게 메모리에 적재되는가?
